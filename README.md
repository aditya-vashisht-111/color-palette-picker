# Color Palette Picker – Colorblind Friendly

A single-purpose micro-tool that generates colorblind-safe color palettes with:

- Light & dark mode chart previews
- Copyable export formats:
  - HEX list
  - CSS variables
  - Matplotlib list
  - Seaborn palette

Built as part of the AI for Bharat – Week 1 Micro-Tools Challenge.

## Features

- Choose palette size (3–8 colors)
- Uses Okabe–Ito style colorblind-safe palette
- Auto-preview charts in light and dark mode
- Copy-to-clipboard buttons for each export format
- 100% frontend – works in any modern browser

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub Pages (for hosting)
